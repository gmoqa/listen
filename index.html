listen

minimal audio transcription tool

Mac/Linux:
brew tap gmoqa/listen
brew install listen

Arch Linux:
yay -S listen

Termux (Android):
curl -sSL https://raw.githubusercontent.com/gmoqa/listen/main/install-termux.sh | bash
  • Local-first: all processing on your device
  • Simple: one command to transcribe
  • Multi-language support (50+ languages)
  • Audio file transcription (mp3, wav, m4a, flac)
  • Auto-stop with silence detection
  • HTTP API server mode
